Battleship-Board-Game

Steps to run this program:

If run on IDE:
1. Run the Server.java
2. Add the address of the server as an argument for the BattleshipGameGUI
3. Follow the instruction in the GUI to play the game.

If run on cmd:
1. Compile both Server.java and BattleshipGameGUI.java
2. Run the Server first
3. Run the BattleshipGameGUI by entering: java BattleshipGameGUI <server_address>
